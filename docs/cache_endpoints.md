# Cache candidate endpoints (API)

Notes:
- public: safe for shared cache/CDN. Use ETag/Last-Modified + Cache-Control public.
- private: user-specific. Cache only on client, use Cache-Control private, must-revalidate.
- short/medium/long indicate suggested max-age (tunable).
- Endpoints not listed are write actions or too sensitive/real-time for caching.
